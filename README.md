# Сервис продажи мерча факультета

Монолитное веб-приложение для продажи мерча, разработанное на Django и PostgreSQL.

## Стек технологий
* **Бэкенд:** Python 3.11, Django
* **База данных:** PostgreSQL
* **Инфраструктура:** Docker, Docker Compose
* **Инструменты:** Git

## Что реализовано
* **Каталог товаров:** Структурированное хранение мерча с фильтрацией по категориям.
* **Архитектура БД:** Спроектирована реляционная структура таблиц в PostgreSQL для управления пользователями, товарами и заказами.
* **Контейнеризация:** Приложение и база данных полностью упакованы в Docker Compose для быстрого запуска и деплоя.

## Инструкция по запуску

1. **Клонировать репозиторий:**
   ```bash
   git clone [https://github.com/твой_юзернейм/название_репозитория.git](https://github.com/твой_юзернейм/название_репозитория.git)
   cd название_репозитория
   ```
2. **Запустить проект через Docker Compose:**
   ```bash
   docker-compose up --build
   ```
